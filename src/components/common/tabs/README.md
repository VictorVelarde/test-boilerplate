# Tabs component

Tabs organize and allow navigation between groups of content that are related and at the same level of hierarchy.

A simple example with no frills.

```html
<Tabs>
  <Tab name="Tab A" selected>
    <div>
      <p>Tab A content</p>
    </div>
  </Tab>

  <Tab name="Tab B">
    <div>
      <p>Tab B content</p>
    </div>
  </Tab>
</Tabs>
```
